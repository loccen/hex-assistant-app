# 诊断包导出说明

应用内“导出诊断包”按钮会调用 `export_diagnostic_package`，将应用数据目录中的配置、校准、日志、截图样本、OCR 回放、捕获记录、报告和缓存打成 zip，输出到应用数据目录的 `reports/` 子目录。

诊断包用于定位截图、OCR、ApexLOL、Overlay 和状态机问题。导出前建议先执行健康检查、截图样本采集、OCR 回放或对应功能操作，让诊断包中保留可复盘材料。

诊断包当前包含：

```text
config/
calibration/
logs/
samples/
ocr-replay/
captures/
reports/
cache/
```

其中 `reports/environment.json` 会记录导出时间、系统平台、应用数据目录、当前可执行文件和安全边界。真实 Windows 验收时还需要人工补充显示器编号、分辨率、缩放、LOL 显示模式、OCR 模型路径、ORT 动态库加载结果和问题描述。

诊断包不得保存敏感令牌，不得包含与本功能无关的个人信息。
