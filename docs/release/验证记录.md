## 发布包说明对齐

当前正式说明以 `docs/05-release-and-diagnostics.md` 和 `release/README.md` 为准：

- 面向用户的 release zip 应固定输出为 `release/hex-assistant-release.zip`，并同步覆盖解压到 `release/hex-assistant-release/`。
- zip 内应只包含 Windows 可执行文件、`WebView2Loader.dll`、`resources/`、`README.txt`、`release-manifest.json` 和 `checksums.txt`。
- `dist/`、`docs/`、Linux installers、deb/rpm/appimage 和源码目录不应作为最终用户包内容。

如果后续重新执行 release 导出，本文件中的产物清单必须按当次真实 zip 清单重写，不能沿用旧记录。
